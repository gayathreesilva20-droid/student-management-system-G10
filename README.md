# student-management-system-G10
Student Management System using Python (Tkinter) with features like grade calculation, fee tracking, and CSV-based data storage.
# Student Management System

# Project Description
This project is a Student Management System developed using Python and Tkinter. It allows users to manage student academic records, calculate grades, track monthly fees, and generate reports efficiently.

# Features
- Add new student records
- View all students in a table
- Search student by ID
- Automatic calculation of average marks
- Grade generation based on performance
- Fee status tracking (Paid/Unpaid)
- Generate individual student reports (CSV)

# Subjects Included

# Technologies Used
- Python
- Tkinter (GUI)
- CSV (Data Storage)

# How to Run
1. Install Python 3
2. Download the project files
3. Run the following command:

python main.py
